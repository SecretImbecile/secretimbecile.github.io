Point Solitude

Game Modification, June 2014

Point Solitude was created for Nolimits 2 Roller Coaster Simulation to further familiarise myself with Blender3D, being the first time I had properly used that software in a realtime graphics application.

The modification was well recieved amongst the game community, especially for its immersive environment, and remains one of the most popular modifications of all time for Nolimits 2 on the Steam Workshop. A second version has been worked on to some extent, a screenshot of both versions is provided below.


Screenshot of the original modification.

The modification was featured by the game developers on several occasions.

Workshop documents for Manchester Raspberry Jam

Community Event

To supplement the taught workshops at Manchester Raspberry Jam, I have also started producing written notes, typeset in LaTeX.

Rather than a set of notes which are worked through during the workshop, these notes aim to provide an alternate form for the workshop material, so that the workshop can be complete by those not at the event, or so that attendees can attempt the exercises again at home, without immediate assistance.

These notes can be viewed below, and their source files can be found on my github page.
